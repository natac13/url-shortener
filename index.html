Natac's Url Shortener

You can pass a URL as a path. As long as the URL is valid you will get back a json response { "originalUrl": [yourInput], shortUrl: [provided] }

Validate URL must have a TLD '.com', '.ca', '.org'. They can omit the protocol 'http', https', 'ftp'

Usage:

Input

hostname/new/http://www.google.ca

Get Back as JSON

{ "originalUrl": http://www.google.ca, shortUrl: hostname/3 }

Then visit:

hostname/3

Redirects to:

http://www.google.ca
